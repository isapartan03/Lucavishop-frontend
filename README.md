# Frontend - Ecommerce Tienda de Ropa

Aplicación frontend para plataforma de comercio electrónico de venta de ropa.

## 🛠️ Tecnologías utilizadas

- React 18
- TailwindCSS
- DaisyUI (componentes UI)
- React Router (navegación)
- Context API (estado global)
- Axios (peticiones HTTP)
- React Hook Form (formularios)

---

## 📋 Requisitos previos

- Node.js (v18 o superior)
- Git

---

## 🚀 Instalación

1. Clonar el repositorio
- git clone https://github.com/TU_USUARIO/ecomerce-frontend.git
- cd ecomerce-frontend

2. Instalar dependencias
- npm install

3. Configurar variables de entorno
- cp .env.example .env

4. Editar el archivo .env:
- REACT_APP_BACKEND_URL=http://localhost:3001/api
- URL_FOR_IMAGES=http://localhost:3001

Ejecutar la aplicación
- npm start

La aplicación correrá en http://localhost:3000

## 📜 Scripts disponibles
- npm start	Modo desarrollo
- npm run build	Generar build para producción
